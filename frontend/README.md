# Medical xAI Web App Frontend

## Stack
- React (Vite)
- Tailwind CSS
- shadcn/ui
- Framer Motion
- Recharts

## Structure
- `src/pages`: Main pages (Dashboard, Upload, Patient, BiasAudit)
- `src/components`: UI components (BodyPartTabs, ExplainabilityPanel, etc.)

## Dev
```bash
npm install
npm run dev
```
